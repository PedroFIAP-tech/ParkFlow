# ParkFlow Visual Direction

## Identidade

ParkFlow deve parecer uma central operacional enterprise para grandes operacoes de estacionamento, frota e sinistros.

Direcao:

- dark mode elegante;
- azul escuro, preto e tons de azul;
- branco para leitura;
- verde, amarelo e vermelho apenas para status;
- cards compactos e operacionais;
- informacao densa, mas limpa;
- pouca decoracao;
- nenhum dashboard grande na home.

## Home / Central Operacional

Prompt para mockup:

```txt
High-fidelity enterprise operations center UI for a product named ParkFlow, premium dark mode, navy blue and black color palette, modern Brasil Park inspired corporate mood, real-time vehicle incident queue, large quick action for new occurrence, search by plate chassis or occurrence ID, compact priority cards, status chips, stopped time indicators, recent operational timeline, AI next-best-action panel, no large dashboard charts, dense but clean B2B SaaS layout, sharp typography, subtle glass panels, 16:9 desktop screen.
```
